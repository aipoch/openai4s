# openai4s
This repository indexes open-source AI for Science (AI4S) projects on GitHub.  Projects are grouped by scientific workflow:  Information Retrieval (e.g., PDF/OCR tools, RAG for literature)  Solution Design (e.g., DeepVariant)  Data Analysis (genomics, biomedical AI agents)  Academic Text (e.g., protein language models, automated poster generation)

---
